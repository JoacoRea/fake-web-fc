# Imágenes de hilos de Threadit

De ahora en más, **vos elegís el nombre del archivo** al subir cada foto —
usalo para describir qué hay en la imagen (jugadores, momento, gol, etc.),
así yo sé exactamente qué estoy referenciando en cada hilo sin adivinar.
Los espacios en el nombre no son problema, los codifico como `%20` al
referenciarlos en el sitio.

## Cómo subir fotos nuevas

1. En GitHub, andá a esta carpeta (`assets/img/threads/`) en el repo `JoacoRea/fake-web-fc`, directo en `main`.
2. **Add file → Upload files**, con un nombre descriptivo de tu elección.
3. Commiteá directo a `main`.
4. Avisame qué foto es (o simplemente el nombre ya lo dice) y yo la engancho al hilo que corresponda.

Mientras una imagen referenciada no exista todavía, el sitio no se rompe —
simplemente no se muestra hasta que la subas.
