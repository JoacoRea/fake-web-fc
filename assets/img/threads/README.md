# Imágenes de hilos de Threadit

Subí acá las capturas/fotos del juego que querés que aparezcan en los hilos.
Los hilos ya están armados esperando estos 4 nombres de archivo exactos
(referenciados en `data/threads.json`):

- `carabao-final-trophy.jpg` — la foto señalando el trofeo (hilo "WE'VE WON IT")
- `carabao-final-ratings.png` — la pantalla de calificaciones con Nico Paz (hilo "Nico Paz's cameo by the numbers")
- `carabao-final-celebration.jpg` — los tres jugadores festejando, camiseta de Grimaldo (hilo "This photo is going straight into Chelsea folklore")
- `galatasaray-3-fingers.jpg` — el festejo con el gesto de "3" dedos (hilo del 3-0 vuelta vs Galatasaray)

## Cómo subirlas

1. En GitHub, andá a esta carpeta (`assets/img/threads/`) en el repo `JoacoRea/fake-web-fc`.
2. **Add file → Upload files**, arrastrá las 4 imágenes con esos nombres exactos.
3. Commiteá directo a `main` (o a una rama y avisame).

Mientras no estén, el sitio no se rompe — las imágenes faltantes simplemente no
se muestran (el layout se ajusta solo).
